Creating a Repository in Github

Signin to Github

In order to create a repository in Github, we have to first Sign-in to Github.
https://github.com


If you are a new user and would like to create an account
  • Click Sign up
  • Enter a desired Username, an Email address and a Password
  • And Sign up.
If you are an existing user
  • Click Sign in
  • Enter Email address or Username
  • And Sign in.
